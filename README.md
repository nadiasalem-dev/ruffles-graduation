# Ruffles Graduation Ceremony (p5.js)

This project is a small animated graduation ceremony built with p5.js.

The goal of the project is to explore animation, storytelling, and accessible design in a creative coding environment. The scene features a graduation stage, audience, announcer captions, and a determined graduate named Ruffles.

Accessibility considerations include:
- color-blind friendly color choices
- captioned announcements
- simple readable visuals

Along the way, observant viewers may notice a few familiar characters in the audience.
